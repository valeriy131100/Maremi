**{discord_prefix}help** - возвращает данный список команд

**{discord_prefix}connect <chat_id>** - подключает сервер к чату ВК. Устанавливает текущий канал как канал по умолчанию

**{discord_prefix}nickname <nickname>** - устанавливает никнейм, который будет использоваться при отправке сообщений в вк

**{discord_prefix}nickname remove** - удаляет никнейм

**{discord_prefix}alias <alias>** - устанавливает алиас для текущего канала. В алиасы можно отправлять сообщения через вк бота с помощью {alias_prefix}<alias>

**{discord_prefix}alias remove <alias>** - удаляет алиас <alias>

**{discord_prefix}default** - устанавливает текущий канал как канал для команды /send

**{discord_prefix}art** - устанавливает текущий канал как канал для команды /art

**{discord_prefix}duplex** - устанавливает текущий канал как канал для дуплекса (режим когда сообщения автоматически отсылаются из ВК в Дискорд и обратно)

**{discord_prefix}split <ref_message>** - разделяет сообщение с вложениями на отдельные. Необходимо переслать с командой оригинальное сообщение

**{discord_prefix}gallery [ref_message|attaches] [n|noninvite]** - превращает текущее или пересланное сообщение в интерактивную галерею. Если передан аргумент noninvite, то создает галерею без сообщения-приглашения.

**{discord_prefix}reload** - перезагружает сообщение из вк (полностью заново генерирует его содержимое)
